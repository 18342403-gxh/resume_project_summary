# 单项目输出示例

## 项目简介

面向新能源车企的智慧出行 B 端 SaaS 平台，基于一码多端架构（H5 / 小程序 / App 内嵌）提供车主全生命周期服务能力，涵盖车辆绑定、充电桩管理、远程控车、售后预约等核心场景，支持多租户灵活配置与白屏化运营，实现业务能力的标准化输出与规模化复制。

## 项目技术

- **跨端架构**：Taro 3 + React，一套代码同时产出微信小程序、H5、App 内嵌 WebView，解决多端一致性与维护成本问题
- **状态管理**：Zustand + React Query，轻量状态方案配合服务端缓存策略，降低全局状态复杂度
- **UI 体系**：基于 NutUI 二次封装业务组件库，统一设计语言与交互规范
- **端侧桥接**：自研 JSBridge SDK，封装 30+ 原生能力调用，提供统一 Promise API
- **工程化**：Monorepo（pnpm workspace）+ Turborepo 构建编排 + ESLint/Prettier 统一规范

## 个人贡献

1. **车辆服务模块**：独立主导车辆绑定、远程控车、车况查询等核心功能的前端实现，覆盖 5 个一级页面与 12 个交互流程，配合后端完成全链路联调与多轮体验优化。
2. **充电业务模块**：端到端负责充电桩地图、扫码充电、充电记录等功能开发，接入高德地图 SDK 并实现自定义聚合渲染，支撑日均 2000+ 次充电场景。
3. **JSBridge 桥接层**：参与桥接 SDK 的能力扩展，新增蓝牙钥匙、推送注册等 8 个原生桥接接口，配合 iOS/Android 团队完成双端适配与兜底方案。
4. **组件库与工程规范**：配合团队推进业务组件沉淀，贡献 6 个高频业务组件（地址选择器、车型选择器等），参与 ESLint 规则定制与 CR 规范落地。
